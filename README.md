# go-wasm-todo-list

* Pure Go ToDo List app running on the browser using the power of WebAssembly.
* This app is implemented in [go-hyperscript](https://github.com/syumai/go-hyperscript/).

## Run App

* https://syumai.github.io/go-wasm-todo-list/

## Environment

* Go 1.12

## Development

```console
make build
```
